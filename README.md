**Revisão:**
Previamente ao início das atividades, realizamos uma breve revisão para recapitular os conceitos abordados na aula anterior.

**Exercício 1: Saudação:**
Neste algoritmo, foquei na interação básica com o usuário. O programa solicita a entrada de um dado textual (nome) e utiliza a técnica de **concatenação** para exibir uma mensagem de boas-vindas personalizada.

**Exercício 2: Soma de Dois Números:**
O objetivo aqui foi trabalhar com operações matemáticas. O fluxo recebe dois valores numéricos, armazena-os em variáveis e realiza o processamento aritmético de adição antes de exibir o total.

**Exercício 3: Verificação de Maioridade:**
Este exercício introduz a **estrutura de decisão**. O algoritmo avalia se a variável `idade` atende à condição de ser maior ou igual a 18. Dependendo do resultado booleano (verdadeiro ou falso), o fluxo segue por caminhos diferentes.

**Exercício 4: Classificação Numérica:**
Aqui utilizei **condicionais compostas** (ou encadeadas). O fluxograma não decide apenas entre duas opções, mas testa se o número é maior que zero, menor que zero ou igual a zero, cobrindo todas as possibilidades matemáticas.

**Exercício 5: Média de Notas:**
Nesta atividade, apliquei a lógica de precedência matemática. O fluxo lê duas notas, soma-as e divide o resultado por 2 para encontrar a **média aritmética**, demonstrando um ciclo completo de Entrada -> Processamento -> Saída.
